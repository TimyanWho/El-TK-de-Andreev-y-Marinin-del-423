# Стоимость проезда железнодорожным транспортом

Практическая работа, вариант 2.

## Что делает приложение
Программа вычисляет стоимость проезда в зависимости от:
- расстояния в километрах;
- количества билетов;
- выбранной комфортабельности.

## Формула расчета
Базовая ставка: **8 руб. за 1 км**.

Коэффициенты:
- плацкарт — **1.0**
- купе — **1.1**
- полулюкс — **1.2**
- люкс — **1.3**

Итог:

**Стоимость = расстояние × 8 × количество билетов × коэффициент комфорта**

## Структура проекта
- `src/RailwayFareCalculator` — WPF-приложение
- `src/RailwayFareCalculator.Core` — общая библиотека с расчетами
- `tests/RailwayFareCalculator.Tests` — автоматизированные тесты xUnit для .NET Framework 4.8
- `docs/Test_Cases_Variant_2.docx` — заполненный тестовый документ

## Как запустить
1. Открыть `RailwayFarePractice.sln` в Visual Studio 2022.
2. Установить .NET 8 SDK.
3. Установить .NET Framework 4.8 Developer Pack.
4. Запустить проект `RailwayFareCalculator` как стартовый.

## Автоматизированные тесты
Тесты покрывают:
- расчет стоимости для всех типов комфорта;
- проверку корректных итоговых значений;
- проверку больших значений;
- обработку невалидных входных данных;
- коэффициент для плацкарта;
- возврат всех полей результата расчета.
